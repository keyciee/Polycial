 🎲 POLYCIAL  The Political Property Game

> *Acquire land. Wield power. Dominate the nation.*

POLYCIAL is a browser-based political property trading game inspired by classic board games, reimagined with a modern UI and strategic gameplay. Players compete to control territories, manage resources, and outmaneuver opponents in a dynamic, turn-based system.



 🚀 Features

- 🎯 Strategic Gameplay — Buy, trade, and develop properties  
- 👥 Multiplayer Support — 2–4 players  
- 🤖 AI Opponents — Optional AI players  
- 🏗️ Property Development — Build houses and hotels  
- 💰 Economic System — Manage assets and rent  
- 🎨 Modern UI/UX — Responsive and clean design  

---

 🕹️ How to Play

1. Start the game and choose number of players  
2. Roll the dice to move  
3. Buy properties and collect rent  
4. Build houses/hotels  
5. Trade with other players  
6. Be the last player with money to win  



 🎮 Controls

| Action | Description |
|--------|------------|
| Roll Dice | Move player |
| End Turn | Next player |
| Build | Upgrade property |
| Mortgage | Get emergency cash |
| Trade | Exchange assets |
